# LinKeeper - Secure Link Manager

A secure link management system with end-to-end encryption.

## Features
- AES-256 encryption for all links
- Strong password requirements
- Login with email or username
- Copy links to clipboard

## Deployment
1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up PostgreSQL
4. Run: `python app.py`
